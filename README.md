# C Projects -- From Zero to Hero

Welcome to the **C Projects From Zero to Hero** repository! This repo
guides you step-by-step through the C programming language, starting
from the absolute basics and gradually moving into more advanced
concepts through practical, hands-on projects.

Whether you're just beginning your C journey or strengthening your
skills, this structured collection will help you grow from **zero ‚Üí
hero**.

1.  Start with **project0(Project_name)**.

## üõ† Requirements

To run the projects, you need:

-   A C compiler (GCC, Clang, or MinGW)
-   A terminal/command prompt
-   Basic command-line usage

Example compile command:

    gcc main.c -o output
    ./output
